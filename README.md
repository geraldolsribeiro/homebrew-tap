# Geraldolsribeiro Tap

## How do I install these formulae?

`brew install geraldolsribeiro/tap/<formula>`

Or `brew tap geraldolsribeiro/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "geraldolsribeiro/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
